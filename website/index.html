ف Persian Transliterator فارسی

Type Persian on Any Keyboard

Persian Transliterator converts your Latin keyboard input into Persian (Farsi) characters as you type — in any web page, offline, and completely private.

Download Extension

Features

  • Real-time Latin → Persian transliteration in standard text fields
  • Global and per-site enable/disable toggles
  • Right-click context menu for on-demand transliteration
  • Modifier-key bypass (Ctrl / Alt / ⌘) to pass Latin characters through
  • Custom mapping editor — tailor the character map to your typing style
  • Offline operation — no data ever leaves your browser

Installation

  1. Download the latest fa-transliterate.zip.
  2. Extract the ZIP — you will find a chrome/ folder inside.
  3. Open Chrome and navigate to chrome://extensions/.
  4. Enable Developer mode (top-right toggle).
  5. Click Load unpacked and select the chrome/ folder.
  6. The extension icon will appear in your toolbar.

Build from Source

git clone https://github.com/cocodedk/fa-transliterate.git
cd fa-transliterate
# Load the chrome/ directory as an unpacked extension in Chrome

No build step required — the extension runs directly from source.

Architecture

chrome/
├── manifest.json       — MV3 manifest, permissions, entry points
├── background.js       — Service worker: context menus, badge
├── content/
│   └── content-script.js — Injected into pages; handles keypresses
├── scripts/
│   └── transliterator.js — Pure transliteration engine (no Chrome APIs)
└── ui/                 — Popup and options pages
