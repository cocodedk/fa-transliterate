ف ترانسلیتریتور فارسی English

تایپ فارسی با هر صفحه‌کلیدی

ترانسلیتریتور فارسی حروف لاتین صفحه‌کلید شما را در لحظه به فارسی تبدیل می‌کند — در هر صفحه وبی، آفلاین و کاملاً خصوصی.

دانلود افزونه

ویژگی‌ها

  • تبدیل آنی لاتین به فارسی در فیلدهای متنی استاندارد
  • کلید روشن/خاموش سراسری و برای هر سایت به‌صورت جداگانه
  • منوی کلیک‌راست برای ترانسلیتریشن دستی
  • دور زدن تبدیل با کلیدهای Ctrl / Alt / ⌘
  • ویرایشگر نگاشت سفارشی — جدول حروف را بر اساس سلیقه تنظیم کنید
  • عملکرد آفلاین — هیچ داده‌ای مرورگر را ترک نمی‌کند

نصب

  1. آخرین نسخه fa-transliterate.zip را دانلود کنید.
  2. فایل ZIP را استخراج کنید — پوشه chrome/ داخل آن قرار دارد.
  3. کروم را باز کنید و به chrome://extensions/ بروید.
  4. حالت توسعه‌دهنده را فعال کنید (کلید بالا-راست).
  5. روی Load unpacked کلیک کنید و پوشه chrome/ را انتخاب کنید.
  6. آیکون افزونه در نوار ابزار ظاهر می‌شود.

ساخت از سورس

git clone https://github.com/cocodedk/fa-transliterate.git
cd fa-transliterate
# پوشه chrome/ را به‌عنوان افزونه بارگذاری کنید

نیازی به مرحله ساخت نیست — افزونه مستقیماً از سورس اجرا می‌شود.
